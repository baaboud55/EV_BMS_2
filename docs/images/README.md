# Image Assets

Place your dashboard screenshots here.

## Recommended images:
- `dashboard_overview.png` - Main dashboard view
- `cell_voltages_chart.png` - Cell voltage chart
- `state_indicators.png` - Status indicators panel

## How to add images:
1. Take screenshots of your BMS dashboard
2. Save them to this folder
3. Update the README.md in the root directory to reference them

Example markdown for images:
```markdown
![Dashboard Overview](docs/images/dashboard_overview.png)
```
